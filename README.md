# avalanchesass_component_hn
Headline classes.

## Install
```
npm install avalanchesass_component_hn --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_component_hn';

/* Import just the mixin file. */
@import 'avalanchesass_component_hn/mixin';
```
